# Changelog

All notable changes to **node-tools** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project aims to follow [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-01-06

### Added

- Initial release with CLI (`nt`) commands:
  - `hash`, `uuid`, `json`, `base64`, `slug`, `time`, `system`
- Programmatic API exports from `src/index.js`
- Node built-in test suite (`node --test`)
- CI workflow for Node.js
